## 🛠️ 技术栈

### 核心框架
- **React 19** - 最新版 React 框架
- **TypeScript** - 类型安全开发
- **Vite** - 快速的构建工具
- **React Router 7.8** - 客户端路由

### UI 框架
- **Ant Design 5.27** - 企业级 UI 组件库
- **Tailwind CSS 4** - 原子化 CSS 框架
- **@ant-design/icons** - 图标库
- **@ant-design/v5-patch-for-react-19** - React 19 兼容补丁

### 数据管理
- **Apollo Client 3.14** - GraphQL 客户端
- **@apollo/client** - 数据状态管理和缓存
- **apollo-upload-client** - 文件上传支持

### Markdown 支持
- **@uiw/react-md-editor** - Markdown 编辑器
- **react-markdown** - Markdown 渲染
- **remark-gfm** - GitHub Flavored Markdown
- **remark-math** / **rehype-katex** - 数学公式支持
- **rehype-highlight** - 代码高亮
- **highlight.js** - 语法高亮引擎
- **github-markdown-css** - GitHub 风格样式

### 图表和可视化
- **mermaid** - 流程图/时序图渲染
- **recharts** - 数据可视化图表库

### 代码生成
- **@graphql-codegen/cli** - GraphQL 代码生成
- **@graphql-codegen/typescript** - TypeScript 类型生成
- **@graphql-codegen/typescript-operations** - 操作类型生成
- **@graphql-codegen/typescript-react-apollo** - Apollo Hook 生成

### 测试框架
- **Vitest** - 单元测试框架
- **@testing-library/react** - React 组件测试
- **@testing-library/jest-dom** - DOM 断言库

### 其他工具
- **dayjs** - 日期时间处理
- **github-slugger** - URL slug 生成
- **react-window** - 虚拟滚动优化

## 🔌 GraphQL API 集成

### API 模块

#### 1. 认证 API (`src/api/graphql/auth.ts`)
```typescript
// Mutations
- REGISTER_MUTATION           // 注册
- LOGIN_MUTATION              // 登录
- EMAIL_LOGIN_MUTATION        // 邮箱登录
- VERIFY_EMAIL_AND_LOGIN_MUTATION  // 验证邮箱并登录
- LOGOUT_MUTATION             // 登出
- REFRESH_TOKEN_MUTATION      // 刷新 Token (新增)
- SEND_VERIFICATION_CODE_MUTATION  // 发送验证码
- VERIFY_EMAIL_MUTATION       // 验证邮箱
- REQUEST_PASSWORD_RESET_MUTATION  // 请求密码重置
- CONFIRM_PASSWORD_RESET_MUTATION  // 确认密码重置
- UPDATE_PROFILE_MUTATION     // 更新个人资料
- CHANGE_PASSWORD_MUTATION    // 修改密码

// Queries
- ME_QUERY                    // 获取当前用户

// Hooks
- useAuth()                   // 认证操作 Hook
- useCurrentUser()            // 当前用户 Hook
```

#### 2. 博客 API (`src/api/graphql/blog.ts`)
```typescript
// Queries
- POSTS_QUERY                 // 文章列表
- POST_QUERY                  // 单篇文章
- SEARCH_POSTS_QUERY          // 搜索文章
- POPULAR_POSTS_QUERY         // 热门文章
- RECENT_POSTS_QUERY          // 最新文章
- TRENDING_TAGS_QUERY         // 热门标签
- POST_VERSIONS_QUERY         // 版本历史 (新增)

// Mutations
- CREATE_POST_MUTATION        // 创建文章
- UPDATE_POST_MUTATION        // 更新文章
- DELETE_POST_MUTATION        // 删除文章
- PUBLISH_POST_MUTATION       // 发布文章
- ARCHIVE_POST_MUTATION       // 归档文章
- LIKE_POST_MUTATION          // 点赞文章
- UNLIKE_POST_MUTATION        // 取消点赞

// Hooks
- usePosts()                  // 文章列表 Hook
- useSearchPosts()            // 搜索文章 Hook
- usePopularPosts()           // 热门文章 Hook
- useRecentPosts()            // 最新文章 Hook
- useTrendingTags()           // 热门标签 Hook
- usePostVersions()           // 版本历史 Hook (新增)
- useLazyPostVersions()       // 懒加载版本历史 (新增)
- useBlogActions()            // 博客操作 Hook
```

## 🤝 贡献指南

### 添加新功能
1. 在 `src/pages/` 或 `src/components/` 中创建新组件
2. 在 `src/api/graphql/` 中定义 GraphQL 操作
3. 在 `src/hooks/` 中创建自定义 Hook
4. 在路由中注册新页面
5. 更新本文档

### 代码规范
- 使用 TypeScript
- 遵循 ESLint 规则
- 组件使用函数式编程
- 使用 Hook 管理状态
- 添加必要的注释
